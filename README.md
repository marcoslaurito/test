# JobberWocky

1. Clone the project
2. Run `npm install`
3. Change `.env` variables if needed
4. Run the app with `npm run start:dev`
5. Run the SEED process sending a request to:

```
http://localhost:3000/api/v1/seed
```

6. Check the swagger in

```
http://localhost:3000/apiDocs
```
